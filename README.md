# Ivallavi Fahrazi Portfolio

A clean, production-ready React portfolio with a modern architecture-inspired layout for international tech recruiters.

## Tech Stack

- React (functional components)
- Vite
- Tailwind CSS

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in terminal (usually `http://localhost:5173`).

## Production Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this project to GitHub.
2. In Vercel, click **New Project** and import the repository.
3. Use defaults:
   - Framework Preset: **Vite**
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Click **Deploy**.

Vercel automatically handles routing for this single-page portfolio.
